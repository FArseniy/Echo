# Superdesign hand-off

The canonical, reusable design specification is [../DESIGN_SYSTEM.md](../DESIGN_SYSTEM.md). Use its token names and component rules when composing Echo or Mirrised screens. Echo is a dark, calm, link-first utility: no dashboards, decorative illustrations, oversized gradients or attention-seeking motion.

Key composition rules: one clear primary action per card, 12–20px internal spacing scale, compact wordmark/header, descriptive labels before controls, mint only for the most important positive action or verified/connected state, and a readable dark surface hierarchy.
