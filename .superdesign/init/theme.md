# Echo — current theme

Echo currently ships one dark theme. The visual direction is quiet nocturnal utility: deep navy surfaces, an ice-mint action color, muted indigo support color and restrained rose for destructive actions. The redesign keeps dark mode as the product default rather than introducing a theme switch or changing user preference behaviour.

The canonical tokens are declared in `public/style.css` under `:root` and documented in the root `DESIGN_SYSTEM.md`.
